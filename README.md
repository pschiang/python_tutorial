##  安裝 python ##
### Anaconda 包 ###
Anaconda 提供了一個完整的 python組合包，包含大多數常用的函式庫，雖然不是每個都立刻用得到，不過可以省很多時間。
官網提供下載 https://www.continuum.io/downloads

先選作業系統，再選版本（請用 python3.x版），接著就是等下載，WINDOWS 和 OSX 基本上是無腦安裝，Linux版本請見官網說明（也夠無腦了）。
